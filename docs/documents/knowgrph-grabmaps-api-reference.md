## Notes (SSOT)

- **Directions coordinate order**: default `lng,lat`; set `lat_first=true` only if passing `lat,lng`.  
  SSOT: https://maps.grab.com/developer/documentation/routes

- **Places coordinate order**: `location` is `"latitude,longitude"` (keyword bias, nearby center, reverse-geo pin).  
  SSOT: https://maps.grab.com/developer/documentation/searching

- **Map rendering adapter policy (knowgrph)**: default to MapLibre GL JS; if MapLibre cannot mount, fall back to GrabMaps Library when BYOK API key is present.
