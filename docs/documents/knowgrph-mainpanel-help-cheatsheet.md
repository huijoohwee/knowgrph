Keep `Key` aligned to the Help cheatsheet text key in source. Keep `Type` aligned to the shared MainPanel Help Icon Library. Keep `Value` concise for the MainPanel Help KTV row. Put longer explanatory copy in `Gesture` or semicolon-separated `Details`.
