# BytePlus · Ark · Video Generation

**Endpoints**

| Method | Path | Operation |
|--------|------|-----------|
| `POST` | `/contents/generations/tasks` | Create a video generation task |
| `GET` | `/contents/generations/tasks/{id}` | Retrieve a video generation task |
| `GET` | `/contents/generations/tasks` | List video generation tasks |
| `DELETE` | `/contents/generations/tasks/{id}` | Cancel or delete a video generation task |

**Async flow:** `POST` returns a task `id` immediately. Poll `GET /contents/generations/tasks/{id}` until `status = succeeded`, then read `content.video_url`. Alternatively, supply `callback_url` in the POST request body to receive server-push notifications on each status transition.

| Token | Canonical model |
|-------|----------------|
| `seedance-2.0` | Dreamina Seedance 2.0 |

| `seedance-1.0-pro-fast` | Seedance 1.0 Pro Fast |
| `seedance-1.0-lite` | Seedance 1.0 Lite (t2v and i2v variants) |
