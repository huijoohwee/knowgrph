## Notes

* All store‑backed numeric UI settings are clamped and persisted via `lsNum/lsSetNum` in `canvas/src/lib/persistence.ts:1–18`.

* `createZoom` respects schema LOD labels; adjusting `performance.lod.hideLabelsBelowScale` changes label visibility under zoom.

* Schema setters provide a comprehensive set of configurable graph behaviors and styles; these can be selectively exposed in the Settings Panel based on product scope.

* Frontend environment overrides use `process.env` and `import.meta.env` for Vite builds; ensure values are injected via `.env` when deploying.

* Backend pipeline configuration relies on environment variables; surface them in docs as reference, not in the frontend Settings Panel.
